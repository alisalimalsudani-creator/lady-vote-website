# Weather Dashboard Project

## Overview
The Weather Dashboard project is designed to provide users with real-time weather information for various locations. Users can search for cities and receive current weather updates, forecasts, and additional details to help them plan their day.

## Features
- Real-time weather data
- Search functionality for cities
- User-friendly interface
- Responsive design
- Integrated API for accurate weather information

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/lady-vote-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd lady-vote-website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## API
The project utilizes [OpenWeather API](https://openweathermap.org/api) for fetching weather information.

## Usage
Simply enter the name of the city in the search bar and press enter. The dashboard will display the current weather information and a 5-day forecast.
